# Java Collections Demos

This folder contains small demonstration programs illustrating the behavior and usage of Java Collections Framework components.

The goal of these examples is to understand:

- internal behavior of common collections
- ordering characteristics
- performance implications
- typical usage patterns

---

## Implementations Covered

Lists

- LinkedListDemo
- ListPolymorphismDemo

Sets

- HashSetDemo
- LinkedHashSetDemo
- TreeSetDemo

Maps

- HashMapDemo
- SimpleHashMap

Sorting

- ComparableDemo
- ComparatorDemo

Queues

- QueueDemo
- PriorityQueueDemo
- ArrayDequeDemo
- LinkedListQueueDemo

---

Each demo focuses on a specific concept of the Java Collections Framework and can be executed independently.
