## Running Core Java Example  
  
> Requires Java 17+ installed and added to PATH.  
  
From project root:  
  
### Compile:  
javac -d bin core-java/oops/payment/*.java  
  
### Run:  
java -cp bin oops.payment.PaymentDemo
