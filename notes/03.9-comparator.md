# Comparator in Java  
  
---  
  
## Overview  
  
Comparator is an interface used to define **custom sorting logic outside a class**.  
  
It belongs to the **java.util** package.  
  
Unlike Comparable, Comparator allows **multiple different sorting strategies** for the same class.  
  
---  
  
## Method Definition  
  
Comparator contains one main method:  
  
    int compare(T o1, T o2)  
  
This method compares two objects.  
  
---  
  
## How compare Works  
  
The compare method returns:  
  
| Return Value | Meaning |
|--------------|--------|
| Negative | First object is smaller |
| Zero | Objects are equal |
| Positive | First object is greater |
  
Example:   
    
    comparator.compare(a, b)  
  
---    
  
## Why Comparator is Needed  
  
Sometimes a class may require **multiple sorting rules**.  
  
Example: Employee objects may need sorting by:  
  
- Salary  
- Name  
- Age  
  
Comparable can define **only one natural order**, but Comparator allows **many different orders**.  
  
---  
  
## Example Comparator  
  
Sorting employees by name:  
  
    Comparator<Employee> nameComparator = new Comparator<Employee>() {  
        @Override  
        public int compare(Employee e1, Employee e2) {    
            return e1.name.compareTo(e2.name);  
        }  
    };  
  
---  
  
## Using Comparator with TreeSet  
  
A comparator can be passed to a TreeSet:  
  
    TreeSet<Employee> employees =  
        new TreeSet<>(nameComparator);  
  
This will sort employees using the comparator rule.  
  
---  
  
## Using Comparator with Collections.sort()  
  
    Collections.sort(list, comparator);  
  
This allows sorting lists with custom logic.  
  
---  
  
## Lambda Comparator (Modern Java)  
  
Java 8 allows writing comparators using lambdas:  
  
    Comparator<Employee> salaryComparator =  
        (e1, e2) -> e1.salary - e2.salary;  
  
This is shorter and cleaner.  
  
---  
  
## Comparable vs Comparator  
  
| Feature | Comparable | Comparator |
|-------|------------|------------|
| Package | java.lang | java.util |
| Method | compareTo() | compare() |
| Sorting Logic | Inside class | Outside class |
| Sorting Rules | One natural order | Multiple custom orders |

---  
  
## Summary  
  
Comparator is used when we need **multiple or flexible sorting strategies**.  
  
It is widely used with:  
  
- TreeSet   
- TreeMap  
- Collections.sort()  
- Streams sorting
